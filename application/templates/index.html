Recipe and inventory database

This database application can be used to keep inventory of your current food ingredients and also as a shared recipe book with ability to search and see what recipes are currently avaible for you.

Using the application

Start by creating an account and adding some ingredients from the common list or create your own. Then you can start browsing avaible recipes and also create them. Recipes can also be browsed (and searched) without logging in.
